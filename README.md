# Q&A Support Bot (RAG System)

This project implements a Retrieval Augmented Generation (RAG) based Q&A bot that answers user questions strictly using content crawled from a website.

## Features
- Website crawling and text cleaning
- Text chunking
- OpenAI embeddings generation
- FAISS vector database for similarity search
- GPT-based answer generation
- FastAPI endpoints
